# Agent Sentinel: Automated Threat Assessment System

## Project Overview

**NOTE:** This project is an independent hackathon submission inspired by and referenced from the **Google Agent Development Kit (ADK) samples.**

Specifically, the core multi-agent architecture, the use of specialized sub-agents, and the iterative workflow pattern are modeled after the following reference project:

> **Source Reference:** [cloude-google/agent-shutton](https://github.com/cloude-google/agent-shutton)

The logic and implementation for the individual agents (e.g., Threat Identifier, Log & Context Agent) are original to this project and adapted for the cybersecurity domain.
